MakeMe Project Resources

The MakeMe project uses a variety of tools and resources to better manage the project, including: source code control and an online issue database.

Source Code Repository

The MakeMe source code is hosted in the GitHub MakeMe Repository. We provide open read-only access. Write access is available to regular contributing developers.

Project Issue Database

The MakeMe open issue and bug list is stored in a GitHub MakeMe Issues database.

Developer Contributors Agreement

Developers are very welcome, both as occasional contributors or core developers. For Embedthis to accept patches or code from any developer, the developer must first have completed the Contributors Agreement. This Agreement is designed to ensure that you are protected and that your patches can be utilized by everyone. All submitted patches that are included in MakeMe are governed by this agreement.
