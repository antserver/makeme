Generating IDE Projects and Makefiles

Bit can generate generate complete project files for building using: make, nmake, Visual Studio, Xcode or plain shell scripts. The generated Makefiles or projects are clean, regular and easy to understand or modify.

Mapping Bit Targets

When Bit generates a Makefile or IDE project, it maps the Bit target onto the best equivalent. In Make, Bit targets become Make targets. The generated Makefiles are flat, meaning that no recursion is used. The entire project is expressed as one Makefile — so it runs fast.

In Visual Studio, Bit creates a Visual Studio and Xcode, sub-projects are created for each Bit executable, library, script or file target.

Generating

Bit uses the --gen command line switch to specify the projects to build. Bit is capable of cross-generating projects for non-native platforms. For example: you can generate an Xcode project for Mac OS X on a Windows system.

bit --continue --platform macosx-x64 -configure . -gen xcode,make

This will generate Xcode and make projects for Mac OS X. The --continue switch indicates that generation should continue even if the required compilers and build tools are not present on the development system.
