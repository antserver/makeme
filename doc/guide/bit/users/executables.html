Executables

Bit creates executables by specifying the required source files and libraries in an exe target. For example: 

bingo: {
    type: 'exe',
    sources: [ 'bingo.c' ],
    depends: [ 'libgame'],
},

Dependencies

Dependent libraries are specified via the depends property. This property takes a list of other targets that must be built before the current target can complete. Note: that these are the target names (target.name property) and not the library name. To specify external libraries that are not built with bit, use the libraries property collection.

libappweb: {
    type: 'lib',
    sources: [ '*.c' ],
    headers: [ '*.h' ],
    depends: [ 'libhttp' ],
    libraries: [ 'pam' ],
},

To do a static link of the executable, set the static property to true. This will link the executable directly with all the required objects. Bit will expand the list of dependant libraries and explicitly link with all their constituent objects.

bingo: {
    type: 'exe',
    sources: [ 'bingo.c' ],
    depends: [ 'libgame' ],
    static: true,
},

To learn more, read about Running Scripts.
