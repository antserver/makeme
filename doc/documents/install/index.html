Installation Overview

Please read the overview information below, then refer to the specific instructions for your operating system. The MakeMe binary installation is available for:

  • Mac OS X Installation Instructions
  • Windows Installation Instructions

For other platforms, you can build from source and then install the built binaries.

Source Code

The MakeMe source code is provided as a separate download distribution. MakeMe is also available via the source code repository for developers who wish to participate in the active development of MakeMe. Developers are welcome!
