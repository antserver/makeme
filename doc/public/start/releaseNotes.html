Release Notes
v0.8.3

Release Notes

Binary releases are supplied for the following operating systems.

  • Windows (Windows-7,8)
  • MAC OS X (10.8+)

Other operating systems may be supported by recompiling MakeMe using the source distribution.

Development Environment Hardware Platform Support

If you wish to build MakeMe from source code, you will need to use a C compiler and the associated development tools. Several development environments are supported. You may choose any of the following to compile and build samples and source code.

  • UNIX/Linux GNU tools
  • Windows Visual Studio 2010 or later

Limitations and Special Notes

  • The MakeMe features and interfaces are still in flux and subject to change. Future releases may change capabilities and break existing builds.
